\# ReNova AI



ReNova AI is an agentic AI modernization platform for converting legacy applications into modern application architectures.



\## POC Goal



The first POC focuses on converting a legacy KST-LCE style VB6 / old C# fault trend analytics module into a modern .NET 8 Web API using:



\- Strands Agents SDK

\- AWS Bedrock AgentCore

\- Python FastAPI

\- Streamlit

\- .NET build and test loop

\- Agentic code conversion and review workflow



\## Core Modules



1\. Legacy Codebase Converter

2\. New Feature / Rewrite Agent



\## Initial POC Scope



The first demo will convert a fake but realistic KST-LCE legacy fault trend module into a modern .NET 8 service with:



\- Domain entities

\- Application service

\- API controller

\- Unit tests

\- Build/test report

\- Migration report
